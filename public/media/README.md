# メディア掲載・登壇の実写真 置き場

トップページの「メディア掲載・登壇実績」セクションは、このフォルダの
以下のファイル名の写真を自動で表示します（置くだけで反映）。
写真が無い間は媒体名のプレートが表示されます。

| ファイル名 | 内容 | 推奨 |
|---|---|---|
| `ivs-launchpad.jpg` | IVS LaunchPad 2026 のピッチ登壇写真 | 横位置 3:2、1200px幅以上 |
| `nttdata-hackathon.jpg` | NTT DATA ハードウェアハッカソン授賞式・デモの写真 | 同上 |
| `dsf-talk.jpg` | DSF 登壇写真（佐藤・柳澤・杉浦） | 同上 |

## 注意
- **自社で撮影した（権利を持つ）写真のみ**を置いてください。
  新聞紙面のスキャンや他者撮影のイベント公式写真は、権利確認が取れるまで不可。
- AI生成画像は使わない方針です。
- 同じ写真を各ニュース記事の `src/data/news/<記事>/heroImage.jpg` にも
  差し替えると、ニュース一覧・記事ページも実写真になります
  （現在は全記事が青い水彩のプレースホルダーのままです）。
